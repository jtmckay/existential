# AI

The AI stack provides local model hosting, chat interfaces, transcription, and text-to-speech — all self-hosted.

| Service | Purpose |
|---|---|
| [Hermes](./hermes) | AI agent gateway + workspace UI |
| [Open WebUI](./open-web-ui) | Chat interface backed by Hermes |
| [Ollama](./ollama) | Local LLM hosting |
| [LightRAG](./lightrag) | GraphRAG over an Obsidian vault, exposed as an MCP tool |
| [Whisper](./whisper) | Speech-to-text transcription |
| [Chatterbox](./chatterbox) | Text-to-speech |
| [ComfyUI](./comfyui) | Node-based image generation |
| [MCP](./mcp) | Playwright browser automation for agents |
