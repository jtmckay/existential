# VPN

- Source: [OpenVPN](https://github.com/OpenVPN/openvpn) / [Tailscale](https://github.com/tailscale/tailscale)
- License: [GPL-2.0](https://www.gnu.org/licenses/old-licenses/gpl-2.0.html) (OpenVPN) / [BSD-3](https://opensource.org/licenses/BSD-3-Clause) (Tailscale)
- Alternatives: WireGuard, ZeroTier, Headscale (self-hosted Tailscale)

Using a VPN allows you to keep your phone and devices connected to all services from anywhere, without exposing services to the open internet. It's the most secure option for remote access.

## Features

- **Encrypted Tunnel**: All traffic between device and home server is encrypted end-to-end
- **No Exposed Ports**: Services stay off the public internet — no attack surface
- **OpenVPN**: Self-hosted, full control, widely supported on all platforms
- **Tailscale**: Zero-config mesh VPN built on WireGuard — easiest setup for personal use
- **Mobile & Desktop**: Works on iOS, Android, macOS, Windows, and Linux clients

## Options

- [OpenVPN](https://openvpn.net/)
- [Tailscale](https://tailscale.com/) (alternative — easier setup)
