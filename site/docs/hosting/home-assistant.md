# Home Assistant

- Source: https://github.com/home-assistant/core
- License: [Apache 2.0](https://www.apache.org/licenses/LICENSE-2.0)
- Alternatives: openHAB, Domoticz, HomeSeer

Home automation platform. See the blog post [Optimizing my life, now with Home Assistant](/blog/optimizing-my-life-home-assistant) for a real-world setup walkthrough.

## Features

- **3,000+ Integrations**: Connects to smart lights, sensors, thermostats, cameras, media players, and more
- **Automations**: Trigger actions based on time, device state, sensor readings, and location
- **Lovelace Dashboards**: Fully customizable visual dashboards with cards and widgets
- **Local Processing**: All automations run on-device — no cloud dependency or outage risk
- **Energy Monitoring**: Track solar generation, grid usage, and per-device power consumption
- **Companion Apps**: iOS and Android apps with location tracking, widgets, and notifications
