# Super Productivity

- Source: https://github.com/johannesjo/super-productivity
- License: [MIT](https://opensource.org/licenses/MIT)
- Alternatives: Vikunja, Todoist, Toggl Track, TickTick
- Status: RIP — unrecoverable sync issues after a few weeks on phone

No need to host this — the Android and Desktop apps use WebDAV for persistence, with Nextcloud acting as the server host.

## Features

- **Integrated Time Tracking**: Track time spent per task with one click
- **Built-In Pomodoro Timer**: Focus sessions with automatic break reminders
- **Jira & GitHub Sync**: Pull tickets directly into your task list
- **Daily Planner**: Schedule tasks to specific days and reorder freely
- **WebDAV Sync**: Persist data across devices via Nextcloud — no dedicated server needed
