# Stirling PDF

- Source: https://github.com/Stirling-Tools/Stirling-PDF
- License: [MIT](https://opensource.org/licenses/MIT)
- Status: RIP — only needed to add text fields, which is doable with OnlyOffice
- Alternatives: Adobe, OnlyOffice

## Features

- **PDF Manipulation**: Merge, split, rotate, reorder, and compress PDFs
- **OCR**: Extract text from scanned documents via Tesseract
- **Annotations & Forms**: Add text, signatures, and fillable form fields
- **Conversion**: Convert PDFs to/from Word, images, HTML, and other formats
- **Watermarking & Stamping**: Add headers, footers, page numbers, and watermarks
- **Fully Local**: No files sent externally; everything processed on your machine
