# Flowise

- Source: https://github.com/FlowiseAI/Flowise
- License: [Apache 2.0](https://www.apache.org/licenses/LICENSE-2.0)
- Alternatives: LangFlow, Dify, n8n
- Status: TBD

GUI for building AI flows.

## Features

- **Visual Flow Builder**: Drag-and-drop canvas for composing LLM pipelines without code
- **300+ Integrations**: Pre-built nodes for LLMs, vector stores, tools, memory, and agents
- **Multi-Provider Support**: OpenAI, Anthropic, Ollama, Hugging Face, Groq, and more
- **API Export**: Deploy any flow as a REST endpoint
- **Document Loaders**: Ingest PDFs, web pages, Notion, and other sources into vector stores
- **Embedded Chat Widget**: Drop a chat interface into any website
