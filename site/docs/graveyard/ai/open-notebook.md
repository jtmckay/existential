# Open Notebook

- Source: https://github.com/lfnovo/open-notebook
- License: [MIT](https://github.com/lfnovo/open-notebook/blob/main/LICENSE)
- Status: TBD
- Alternatives: Google Notebook LM, Obsidian, Notion, Logseq

## Features

- **Privacy-First Research**: Local storage, no cloud dependencies
- **Multi-Modal Content**: Import PDFs, videos, audio, web pages, Office documents, and more
- **AI Provider Flexibility**: 16+ providers including OpenAI, Anthropic, Ollama, Google, Groq, and more
- **Professional Podcast Generation**: Multi-speaker podcast creation with customizable profiles
- **Intelligent Search**: Full-text and vector search across all research content
- **Context-Aware AI Chat**: Conversations powered by your specific research materials
- **Citations & Source Tracking**: AI answers with proper source citations
