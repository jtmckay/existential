# Linkwarden

- Source: https://github.com/linkwarden/linkwarden
- License: [AGPL-3](https://www.gnu.org/licenses/agpl-3.0.html)
- Alternatives: Raindrop.io, Wallabag, Shiori, Hoarder
- Status: RIP — no eject, and annoying to load hundreds of bookmarks

## Features

- **Bookmark Archiving**: Automatically snapshots saved pages (screenshot + full HTML)
- **Collections & Tags**: Organize bookmarks into hierarchical collections with tags
- **Full-Text Search**: Search across all saved page content and metadata
- **Collaborative Sharing**: Share collections with other users or make them public
- **Browser Extension**: One-click saving from Chrome, Firefox, and other browsers
