# Fabric

- Source: https://github.com/danielmiessler/fabric
- License: [MIT](https://opensource.org/licenses/MIT)
- Alternatives: LLM CLI, PromptLayer, custom shell scripts
- Status: TBD — reference prompts only

LLM prompt organizer and facilitator. The CLI tool could be useful, but a couple hours into trying to get a GUI for browsing the prompts led to giving up. The prompts are better treated as generic knowledge, searchable and usable like the rest of personal notes.

## Features

- **Curated Pattern Library**: 100+ expert-crafted prompts for summarizing, extracting insights, writing, coding, and more
- **CLI-First**: Pipe any text through a pattern — e.g. `pbpaste | fabric --pattern summarize`
- **Composable Pipelines**: Chain patterns together for multi-step processing
- **Multi-Provider Support**: OpenAI, Anthropic, Ollama, Google, and others
- **Custom Patterns**: Add your own prompts alongside the built-in library
