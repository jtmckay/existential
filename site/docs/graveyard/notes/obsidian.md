## File Sync for Phone

1. Install Obsidian app
2. Install Nextcloud app
3. Install FolderSync app (if Obsidian can't access Nextcloud files directly)
4. Set up two-way sync between a Nextcloud vault directory and the directory accessible by the Obsidian app
