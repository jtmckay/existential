# Obsidian

- Source: https://obsidian.md (proprietary)
- License: Proprietary (free for personal use)
- Alternatives: Logseq, Notion, Roam Research, Joplin
- Status: RIP — not open source, but top choice if FOSS isn't a requirement

## Features

- **Local-First Markdown**: Notes stored as plain `.md` files — no proprietary format
- **Bidirectional Linking**: `[[wiki-style]]` links with a knowledge graph visualization
- **Canvas**: Freeform visual board for arranging and connecting notes
- **Rich Plugin Ecosystem**: 1,000+ community plugins (Excalidraw, Dataview, Templater, etc.)
- **Themes & Customization**: Extensive CSS theming and layout options
- **WebDAV Sync**: Works with Nextcloud for self-hosted mobile sync

## Core Plugins Used

- Unique note creator

## File Sync for Phone

1. Install Obsidian app
2. Install Nextcloud app
3. Install FolderSync app (if Obsidian can't access Nextcloud files directly)
4. Set up two-way sync between a Nextcloud vault directory and the directory accessible by the Obsidian app
