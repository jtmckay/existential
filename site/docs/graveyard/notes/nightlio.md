# Nightlio

- Source: https://github.com/shirsakm/nightlio
- License: [MIT](https://opensource.org/licenses/MIT)
- Alternatives: Daylio, Standard Notes, Diarium
- Status: RIP — had it working building locally, broken with Docker images

Daily check-in / journaling app.

## Features

- **Daily Mood Tracking**: Log emotional state with ratings and emoji
- **Short Journal Entries**: Capture brief notes alongside each check-in
- **Calendar View**: Review past entries by date at a glance
- **Minimalist Interface**: Distraction-free design focused on the daily ritual
- **Self-Hostable**: Deploy via Docker with a local database
