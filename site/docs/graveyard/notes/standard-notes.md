# Standard Notes

- Source: https://github.com/standardnotes/server
- License: [AGPL-3](https://www.gnu.org/licenses/agpl-3.0.html)
- Alternatives: Logseq, Joplin, Bear, Obsidian
- Status: RIP — sync issues, and doesn't support note hyperlinking (graph, backlinking)

## Features

- **End-to-End Encryption**: All notes encrypted client-side by default
- **Cross-Platform**: Web, desktop (Mac/Windows/Linux), and mobile (iOS/Android) apps
- **Editor Extensions**: Markdown, rich text, code, spreadsheet, and task editors
- **Version History**: Restore any previous version of a note
- **Self-Hosted Sync Server**: Full data ownership with your own server
