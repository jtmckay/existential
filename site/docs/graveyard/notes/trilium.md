# Trilium

- Source: https://github.com/TriliumNext/Notes
- License: [AGPL-3](https://www.gnu.org/licenses/agpl-3.0.html)
- Alternatives: Logseq, Obsidian, AppFlowy, Joplin
- Status: RIP — lacks proper markdown support

## Features

- **Hierarchical Note Tree**: Organize notes in a deeply nested tree structure
- **Note Attributes & Relations**: Tag notes with key-value attributes and link them relationally
- **Scripting**: Automate and extend behavior with JavaScript-based scripts
- **Note History**: Full version history with diff view for every note
- **Self-Hosted**: Web interface plus an optional desktop app (sync server built-in)
