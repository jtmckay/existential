# NodeRED

- Source: https://github.com/node-red/node-red
- License: [Apache 2.0](https://www.apache.org/licenses/LICENSE-2.0)
- Alternatives: Decree, n8n, Home Assistant automations
- Status: RIP — primary use case is IOT, not general automation

## Features

- **Browser-Based Visual Editor**: Wire flows by connecting node boxes
- **Large Node Ecosystem**: Thousands of community nodes for IoT, APIs, databases, and protocols
- **MQTT & IoT Native**: First-class support for MQTT, serial, and hardware integrations
- **JavaScript Functions**: Drop into code at any point in a flow
- **Low Resource Usage**: Runs on Raspberry Pi and other low-powered hardware
- **Flow Library**: Share and import flows from the public Node-RED library
