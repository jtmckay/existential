# Lowcoder

- Source: https://github.com/lowcoder-org/lowcoder
- License: [AGPL-3.0](https://github.com/lowcoder-org/lowcoder/blob/main/LICENSE)
- Alternatives: Appsmith, Retool, Tooljet
- Status: As needed — better suited for customer-facing / external apps (prettier UI)

Open-source low-code platform for building internal and customer-facing apps with minimal code. Continues from the abandoned Openblocks project.

## Features

- **Visual UI Builder**: 120+ built-in components with drag-and-drop editor
- **Native Embedding**: Embed apps directly in websites without iframes
- **Data Connections**: PostgreSQL, MongoDB, MySQL, Redis, REST API, and more
- **WebSocket Support**: Real-time data updates
- **Reusable Modules**: Shared query library and component modules across apps
- **JavaScript Throughout**: Escape hatch to JS anywhere in the builder
- **RBAC**: Role-based access control
- **Version History**: Auto-saved with full version history
- **App Theming**: Theme editor for consistent styling
- **Self-Hosted**: Full control, or free community cloud option
