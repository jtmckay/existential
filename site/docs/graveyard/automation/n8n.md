# N8N

- Source: https://github.com/n8n-io/n8n
- License: [Sustainable Use License](https://github.com/n8n-io/n8n/blob/master/LICENSE.md) (non-commercial restrictions)
- Alternatives: ActivePieces, Decree, Zapier, Make
- Status: RIP — closed source licensing practices

For licensing reasons, this cannot be used to automate anything for anyone else. Must be self-hosted and used personally to remain free.

## Features (when it was in use)

- Visual workflow builder with code escape hatches
- Postgres for workflow state and chat history
- Qdrant vector database for RAG
