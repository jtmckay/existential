- Alternatives: Airflow, Prefect, n8n, Windmill
- Status: TBD

## Features

- **YAML-Based Workflows**: Define pipelines as code with version control
- **500+ Plugins**: Built-in integrations for databases, cloud providers, APIs, and messaging
- **Event-Driven Triggers**: Webhooks, schedules, file events, and message queue triggers
- **Real-Time Dashboard**: Live workflow monitoring, logs, and execution history
- **Scalable Architecture**: Runs on a single node or distributed across a cluster
- **Namespace Isolation**: Organize workflows into namespaces for multi-tenant environments
