# Appsmith

- Source: https://github.com/appsmithorg/appsmith
- License: [Apache 2.0](https://github.com/appsmithorg/appsmith/blob/release/LICENSE)
- Alternatives: Lowcoder, Retool, Tooljet
- Status: RIP — replaced by Lowcoder; better for internal apps but less polished UI

Open-source low-code platform for building dashboards, admin panels, and internal tools.

## Features

- **Low-Code App Builder**: Drag-and-drop interface for dashboards and admin panels
- **25+ Database Integrations**: Connect to Postgres, MySQL, MongoDB, S3, and more
- **API Connectivity**: REST and GraphQL support
- **IT Automation**: Workflow and service management tooling
- **Agentic AI**: Built-in AI agent platform
- **Self-Hosted**: Docker, Kubernetes, AWS AMI, or Appsmith Cloud
